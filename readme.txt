This minimalistic WordPress plugin displays great hand picked quotes in your blog's sidebar.

== Description ==

Displays great inspirational hand picked quotes in your blog's sidebar, daily or randomly. Your visitors can tweet quote with single click. You can choose which quotes to display which day, you can edit and add your own quotes. There more than 250 unique quotes. To install, click activate and then go to Appearance > Widgets to find 'Simple Daily Quotes'. All settings are contained within the widget. Visit plugin homepage for help about editing/adding quotes.

== Features ==

::Simplicity:: Simple installation and settings. Just add the widget to your sidebar, set options and plugin is ready to use.
::Quality quotes:: Database contain more than 250 unique hand-selected quotes.
::Add/Change quotes:: You can choose which quote you want to have on which day. You can change edit, delete or add new quotes easily.
::Tweet quotes:: Visitor of your blog can tweet loaded quote with single click.
::No duplicate quotes:: There are more than 250 quotes, for each day per year. No duplicated quotes.

== Installation ==

1. Upload the file `SimpleDailyQuotes.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets and drag 'Simple Daily Quotes' to your sidebar
4. Update the settings in the widget: Widget title, Load quotes, Visitors can tweet, etc.


== Frequently Asked Questions ==

Settings can be edited from within the widget itself, here are the options:

= "Widget title" Option: =
This option will change the front-end title of your widget, it defaults to "Simple Daily Quotes"

= "Load quotes" Option: =
This option allows you to either load the quote once per day, or have it display a random quote each time your blog is viewed.
